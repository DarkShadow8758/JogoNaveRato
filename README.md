# Jogo Nave Rato 🚀🐭

Um jogo desenvolvido no Unity, onde o jogador controla uma nave enfrentando desafios espaciais.

## 🛠️ Tecnologias Utilizadas
* **Unity** (Versão Recomendada: 2022.3.62f3)
* **Visual Scripting** (Unity Graph)
* **TextMeshPro** para a interface do usuário (HUD)

## 🎮 Funcionalidades (HUD)
O projeto conta com um sistema de HUD dinâmico que exibe:
* **Cronômetro:** Tempo de jogo formatado em `MM:SS`.
* **Contador de Itens:** Exibe a quantidade de itens coletados pelo jogador.
* **Instruções de Vitória:** Texto dinâmico que orienta o jogador sobre o objetivo para vencer.
Sistema de Geração de Inimigos:
* **Inimigo1:** Se movimenta de forma aleatória da direita para esquerda

## 🚀 Como executar o projeto

### Pré-requisitos
Certifique-se de ter o **Unity Hub** instalado em sua máquina.

### Passo a passo
1.  **Clonar o repositório:**
    ```bash
    git clone [https://github.com/DarkShadow8758/JogoNaveRato.git](https://github.com/DarkShadow8758/JogoNaveRato.git)
    ```
2.  **Abrir no Unity:**
    * Abra o *Unity Hub*.
    * Clique em **Add** e selecione a pasta da raiz do projeto.
    * Selecione a versão do editor instalada (recomenda-se a 2021.3.x ou mais recente).
3.  **Configurar dependências:**
    * Caso apareça um aviso de "TextMeshPro", clique em **Import TMP Essentials** na janela que abrir automaticamente.
4.  **Jogar:**
    * No painel `Project`, procure pela pasta `Scenes`.
    * Abra a cena principal (`TestNight`).
    * Clique no botão **Play** no topo do editor.

## 🕹️ Controles
* **Movimentação:** Teclas `W`, `A`, `S`, `D` ou Setas.
